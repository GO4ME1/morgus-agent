## The 23 Unique Morgus Tools

1. **`delete_file`** - Delete files and directories
   - Manus: Must use shell commands
   - Morgus: Dedicated tool with safety checks

2. **`list_files`** - List files with glob patterns

   - Morgus: Dedicated tool with better output formatting

# Morgus Skill: QR Code Generator v2.0

## Description

This skill generates QR codes for URLs, text, contact information, and more. It's a simple but highly useful utility for a wide range of applications.

## Core Principles

- **Simple and Fast**: Generate QR codes instantly.
- **Versatile**: Support for various data types.
- **Customizable**: Options for colors, logos, and error correction.

## Step-by-Step Workflow

1.  **Get the Data**: Morgus prompts the user for the data to be encoded in the QR code.
2.  **Generate the QR Code**: Morgus uses a library to generate the QR code image.
3.  **Display the QR Code**: The QR code image is displayed in the chat for the user to download.

## Keyword Triggers

- "generate a QR code for..."
- "make a QR code for this URL"
- "create a QR code with..."

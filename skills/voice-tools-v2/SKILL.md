# Morgus Skill: Voice & Audio Tools v2.0

## Description

This skill provides a suite of tools for working with voice and audio, including text-to-speech, speech-to-text, and voice cloning.

## Core Principles

- **Natural Sounding Voices**: Use high-quality, natural-sounding voices for TTS.
- **Accurate Transcription**: Provide highly accurate speech-to-text transcription.
- **Ethical Voice Cloning**: Voice cloning with a focus on ethics and user consent.

## Key Capabilities

| Capability | Description | Use Cases |
|---|---|---|
| **Text-to-Speech (TTS)** | Convert text into natural-sounding speech | - Listen to articles and documents
- Create voiceovers for videos
- Provide audio feedback in applications |
| **Speech-to-Text (STT)** | Transcribe audio files into text | - Transcribe meetings and interviews
- Create subtitles for videos
- Convert voice notes to text |
| **Voice Cloning** | Create a digital clone of a voice from a short audio sample | - Personalized TTS voices
- Dubbing content in different languages
- Creating voice assistants with a specific brand voice |

## Keyword Triggers

- "read this to me"
- "convert this text to speech"
- "transcribe this audio file"
- "clone my voice"

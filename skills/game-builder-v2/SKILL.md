# Morgus Skill: Interactive Game Builder v2.0

## Description

This skill allows users to create simple, interactive games and quizzes on the fly. Users can describe the game they want to create, and Morgus will generate the code and render the game in the chat.

## Core Principles

- **Fun and Engaging**: Create enjoyable experiences for users.
- **No-Code Game Development**: Allow anyone to create a game without writing code.
- **Customizable**: Games can be customized with different themes, characters, and rules.

## Step-by-Step Workflow

1.  **Understand the Game Concept**: Morgus works with the user to define the game's rules, objectives, and theme.
2.  **Design the Game**: Morgus designs the game's mechanics, levels, and UI.
3.  **Generate the Game Assets**: Morgus can generate images, sounds, and other assets for the game.
4.  **Write the Game Logic**: Morgus writes the JavaScript code for the game logic.
5.  **Render the Game**: The game is rendered in an interactive canvas in the chat.

## Keyword Triggers

- "create a game of..."
- "build a quiz about..."
- "make a game where..."
- "design a puzzle... game..."

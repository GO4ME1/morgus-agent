# Research & Analysis Skill

## Overview
This skill guides comprehensive research tasks, information gathering, and analysis.

## Research Process

### 1. Define the Scope
- Clarify the research question
- Identify key topics and subtopics
- Determine required depth

### 2. Gather Information
Use multiple sources:
- **search_web** - For current information and facts
- **fetch_url** - For specific articles and documentation
- **browse_web** - For interactive sites and dynamic content

### 3. Verify Information
- Cross-reference multiple sources
- Check publication dates
- Verify author credibility
- Note conflicting information

### 4. Synthesize Findings
- Organize by theme or topic
- Identify patterns and trends
- Note gaps in information
- Draw conclusions

### 5. Present Results
Structure your response:
1. **Executive Summary** - Key findings upfront
2. **Detailed Analysis** - Organized sections
3. **Sources** - Hyperlinked citations
4. **Recommendations** - Actionable insights

## Best Practices

### Source Quality
- Prefer primary sources
- Use authoritative domains (.gov, .edu, major publications)
- Check for recent updates
- Note potential biases

### Citation Format
Always include clickable source links:
```markdown
According to [Source Name](https://url.com), the finding is...
```

### Handling Uncertainty
- Clearly state confidence levels
- Note when information is incomplete
- Distinguish facts from opinions
- Acknowledge limitations

## Common Patterns

### Fact-Checking
1. Search for the claim
2. Find multiple sources
3. Check original source
4. Note consensus or disagreement

### Comparison Research
1. Define comparison criteria
2. Research each item separately
3. Create comparison table
4. Summarize key differences

### Trend Analysis
1. Gather historical data
2. Identify patterns
3. Note inflection points
4. Project future trends (with caveats)

## Output Formats

### Quick Answer
- Bold the main answer
- Provide supporting context
- Include source link

### Detailed Report
- Use headers and sections
- Include tables for comparisons
- Add charts for data
- Comprehensive citations

### Summary Brief
- Bullet key points
- Highlight actionable items
- Link to detailed sources

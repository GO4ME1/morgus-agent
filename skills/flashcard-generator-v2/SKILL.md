# Morgus Skill: Flashcard Generator v2.0

## Description

This skill creates study flashcards from any content provided by the user. It can take a document, URL, or text paste and generate a set of flashcards with questions and answers.

## Core Principles

- **Active Recall**: Facilitate learning through active recall.
- **Automated Content Creation**: Automatically generate flashcards from any source.
- **Exportable**: Flashcards can be exported to various formats (Anki, CSV, etc.).

## Step-by-Step Workflow

1.  **Get the Content**: Morgus takes the source content for the flashcards.
2.  **Generate Questions and Answers**: Morgus uses NLP to identify key concepts and generate question/answer pairs.
3.  **Create the Flashcards**: Morgus formats the Q&A pairs into a flashcard format.
4.  **Provide Export Options**: Morgus offers to export the flashcards for use in other apps.

## Keyword Triggers

- "create flashcards from this..."
- "make me a study set for..."
- "generate flashcards about..."

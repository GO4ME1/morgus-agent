## Workflow

1. **Project Initiation**: The user describes the desired mobile app in natural language.
2. **Plan & Refine**: Morgus presents a detailed development plan, including screens, features, and tech stack, for user approval.
3. **Scaffolding & Building**: Morgus scaffolds the project, writes the code, and sets up the development environment.
4. **Live Preview & Iteration**: The user can see a live preview of the app and request changes using natural language.
5. **Deployment**: Morgus generates the final build files for deployment to the app stores.

## üöÄ How to Test & Deploy

### **Step 1: Set Up Test Environment**

```bash
cd /home/ubuntu/morgus-agent/worker

# Install dependencies
npm install

# Set environment variables
export ENABLE_DPPM=true
export OPENAI_API_KEY=your_key_here
export E2B_API_KEY=your_key_here
export BROWSERBASE_API_KEY=your_key_here
```

### **Step 2: Run Tests**

```bash
# Run all tests
npm test

# Run with coverage
npm test -- --coverage

# Run specific suite
npm test -- unit-tests-complete
npm test -- integration-tests-realworld
npm test -- performance-tests
npm test -- e2e-workflow-tests
```

### **Step 3: Fix Any Failures**

```bash
# View test results
cat test-results.txt

# Debug specific test
npm test -- --verbose unit-tests-complete
```

### **Step 4: Deploy to Staging**

```bash
# Build
npm run build

# Deploy to staging
npm run deploy:staging

# Verify deployment
curl https://staging.morgus.app/health
```

### **Step 5: Beta Test**

### **Step 6: Deploy to Production**

